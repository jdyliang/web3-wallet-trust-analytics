# Feature Definitions

This document describes the behavioral features engineered from wallet transaction data.

## 1. wallet_age_days
Number of days between the first and last observed transaction.

## 2. tx_count
Total number of transactions made by the wallet.

## 3. avg_tx_value
Average transaction value.

## 4. tx_frequency
Average number of transactions per active day.

## 5. active_days
Number of days with at least one transaction.

## 6. burst_ratio
Proportion of transactions occurring in short bursts (high activity within a short time window).

## 7. stablecoin_tx_count
Number of transactions involving stablecoins.

## 8. stablecoin_tx_ratio
Proportion of stablecoin transactions relative to total transactions.

## 9. unique_counterparties
Number of unique addresses interacted with.

## 10. activity_consistency
Measure of how evenly transactions are distributed over time.
